# Visual Asset Skill — AI / 3D / Graphics (2026)

Ти не просто верстаєш сторінку.

Ти створюєш ВІЗУАЛЬНУ МОВУ продукту.

---

## ❗ Головне правило

НЕ використовуй:
- стокові картинки
- placeholder
- іконки з інтернету

Замість цього:
→ генеруй або симулюй власні visual assets

---

## 🎯 Що потрібно створити

1. Hero visual (ОБОВ’ЯЗКОВО)
2. Background visuals
3. Section graphics
4. Icon system
5. Logo (якщо слабкий — перероби)

---

## 🧠 Hero Visual

Зроби один з варіантів:

### ВАРІАНТ 1 (рекомендовано)
- abstract AI system
- nodes + connections
- glowing points
- animated graph

### ВАРІАНТ 2
- floating UI cards
- interconnected modules
- system layers

---

## 🌌 Background Visuals

- gradient blobs
- radial glow
- animated noise
- soft moving layers

---

## 🧊 Glass / UI Elements

- glass cards
- blur overlays
- translucent panels
- light reflections

---

## 🧠 Section Visuals

Для кожного блоку:

- Architecture → layers visualization
- Modules → card system
- Timeline → animated flow
- Roadmap → progression

---

## 🎬 Animation (КРИТИЧНО)

- floating animation
- glow pulse
- movement
- parallax

---

## 🎨 Logo (ВАЖЛИВО)

Якщо логотип виглядає примітивно:

→ перероби його

Новий логотип:

- мінімалістичний
- технологічний
- може бути:
  - abstract symbol
  - geometric mark
  - AI / flow символ

НЕ:
- текст без стилю
- простий значок

---

## 🧩 Icons

- line або soft 3D
- єдиний стиль
- не стандартні бібліотеки

---

## 🔥 Depth

Кожен visual має:

- glow
- blur
- shadow
- layered feel

---

## ❌ Заборонено

- пусті секції без графіки
- текст без візуалу
- прості картинки
- дешеві градієнти

---

→ реальний AI продукт

→ система
